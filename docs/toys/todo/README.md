# TODO
[なんのひねりもないTODOアプリ](https://inabajunmr.github.io/vue-playground/toys/todo/todo.html)

# 仕様
 * card
   * タイトルを編集できる
   * taskを追加できる
   * taskは複数持てる
   * card自体も複数定義できる
   * 消せる
 * task
   * テキストを編集できる
   * 完了ステータスを編集できる
   * 消せる
